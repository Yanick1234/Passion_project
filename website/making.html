  • Inspiratie opdoen

    Bij het bedenken van de product(en) die ik ga opleveren bij mijn passion project dacht ik gelijk aan een game. Doormiddel van deze game kan ik mijn designing skills laten zien en kan ik meer oefenen aan mijn programmeer vaardigheden. Ik ga naast de game ook een website bouwen waarop de game speelbaar is.

    Om inspiratie op te doen naar de game zowel de website heb ik de volgende eisen gesteld: Pixel-art, Retro-style, Indie-game. Hierbij heb ik de volgende games uitgekozen: Starbound Stardew Valley Dead Cells

    Van deze games heb ik naar de visuals gekeken en de core mechanics en gameplay van de game. Ook heb ik naar de websites gekeken en heb ik hier inspiratie uit opgedaan.

  • Concept bedenken

    Nadat ik inspiratie had opgedaan wilde ik zo snel mogelijk aan de slag gaan met schetsen en designen. Echter is een uitgewerkt concept belangrijk. Door deze op te stellen heb je een duidelijke gedachte waar je naartoe wilt gaan met de game en de website.

    De game: De speler bevindt zich in een kerker. In deze kerker zijn verschillende gevaren. De speler moet deze gevaren zien te overkomen. Dit doet de speler doormiddel van zijn flexibiliteit, je bent snel en kan onwijs hoog springen.

    De website: De website is een hulpmiddel om naar het spel te navigeren. Op de website is terug te vinden waar het spel over gaat en hoe het werkt. Verder is het proces van de ontwikkeling terug te lezen en de website is belangrijk voor de branding van het spel.

  • Designen

    Ik heb in dit project veel ge-designed. Zo ben ik aan de slag gegaan met een tile-map van het spel en heb ik hierbij basis ruimtes gemaakt om te kijken hoe het spel eruit zal zien. Denk hierbij aan een kamer uit een kerker. Hierbij heb ik ook de speler gemaakt met zijn verschillende animaties. En heb ik de schetsen omgezet in digitale designs.

  • realiseren

    Om de game te realiseren heb ik gebruik gemaakt van Phaser 3. Phaser 3 is een game framework dat volledig draait in javascript. Doormiddel van tutorials heb ik een idee gekregen hoe dit framework werkt en heb ik dit als basis gebruikt om zo de game te realiseren.

    De website heb ik gemaakt doormiddel van HTML, CSS en javascript. Ik wil mij in dit project vooral focussen op de game en heb daarom gekozen voor het bouwen van een simpele website.
